# 로또 번호 추천 시스템 실행 가이드

## 빠른 시작

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 프로그램 실행

```bash
python main.py
```

### 3. 대시보드 실행 (선택)

```bash
streamlit run dashboard_app.py
```

---

## 메뉴 안내

| 번호 | 기능 |
|------|------|
| 1 | 당첨번호 분석 (빈도·합계·홀짝·구간·연속) |
| 2 | 통계 기반 번호 추천 |
| 3 | 제외번호 관리 |
| 4 | 고급 추천 (제외/고정 번호 설정) |
| 5 | 저장된 조합 관리 |
| 6 | 고유 패턴 추천 (AI + 미출현) |
| 7 | 최신 데이터 수집 / 자동 업데이트 |
| 8 | 종합 패턴 분석 추천 (10개 지표) |
| 0 | 종료 |

---

## 데이터 파일

- **`로또당첨번호.csv`** — 당첨번호 데이터 (메뉴 7번으로 자동 수집)
- **`saved_numbers.json`** — 저장된 번호 조합
- **`exclude_numbers.json`** — 제외번호 목록
- **`ai_models.pkl`** — 학습된 AI 모델 (메뉴 6번 최초 실행 시 자동 생성)

---

## 테스트 실행

```bash
pytest
```

---

## 문제 해결

| 증상 | 해결 |
|------|------|
| `ModuleNotFoundError` | `pip install -r requirements.txt` 재실행 |
| 데이터 없음 경고 | 메뉴 7번으로 데이터 수집 후 재시도 |
| AI 모델 무결성 오류 | `ai_models.pkl` 삭제 후 메뉴 6번 재실행 |
| CSV 인코딩 오류 | 파일을 UTF-8로 저장되어 있는지 확인 |
