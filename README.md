# 🎲 로또 번호 추천 시스템 v3.0 (DNN + AI 패턴 학습)

Deep Neural Network(DNN)와 AI 패턴 학습을 활용한 로또 번호 추천 시스템입니다. 기존의 통계적 분석, 패턴 매칭, 강화학습과 함께 DNN 예측 기능과 AI 패턴 학습 기능을 제공합니다.

## 🚀 주요 기능

### 1. 🌐 당첨번호 수집 (신규)
- **자동 데이터 수집**: 동행복권 사이트에서 당첨번호 자동 수집
- **최신 데이터 업데이트**: 새로운 회차 데이터 자동 업데이트
- **범위별 수집**: 특정 회차 범위 선택 수집
- **CSV 파일 저장**: 수집된 데이터를 CSV 형식으로 저장
- **중복 방지**: 기존 데이터와 중복되지 않는 데이터만 추가

### 2. 기존 분석 기능
- **통계적 분석**: 번호 빈도, 간격 패턴, 합계 범위 분석
- **패턴 매칭**: 홀짝 패턴, 연속번호 패턴, 구간별 분포 분석
- **앙상블 분석**: 여러 분석 방법을 조합한 통합 분석
- **강화학습(RL)**: 학습된 정책을 기반한 번호 선택

### 3. 🧠 DNN 기반 예측
- **Deep Neural Network 모델**: 시계열 데이터 기반 번호 예측
- **고급 특성 생성**: 번호 빈도, 통계적 특성, 패턴 정보 활용
- **신뢰도 기반 예측**: 각 예측에 대한 신뢰도 점수 제공
- **모델 성능 평가**: MSE, MAE, 정확도 등 다양한 지표
- **훈련 히스토리 시각화**: 손실 및 성능 그래프 제공

### 4. 🤖 AI 패턴 학습 (신규)
- **회차 그룹핑**: 복수의 회차를 그룹으로 나누어 패턴 분석
- **클러스터링**: K-means를 사용한 패턴 클러스터링
- **머신러닝 모델**: RandomForest, GradientBoosting을 활용한 예측
- **높은 확률 조합**: AI 모델이 학습한 패턴을 기반으로 높은 확률의 조합 생성
- **모델 저장/로드**: 훈련된 모델을 파일로 저장하고 재사용

### 5. 🔢 합계 패턴 분석 (신규)
- **합계 통계 분석**: 과거 당첨번호의 합계 분포 및 통계 분석
- **구간별 분류**: 낮음(70-120), 보통(121-150), 높음(151-180), 매우높음(181-210)
- **트렌드 예측**: 최근 20회의 합계 변화 패턴을 기반으로 다음 회차 예측
- **균형 잡힌 조합**: 홀짝, 고저, 연속번호 균형을 고려한 번호 조합 생성
- **점수 기반 평가**: 각 추천 조합에 대한 종합 점수 제공

### 6. 번호 관리 기능
- **번호 저장**: 추천받은 번호를 저장하고 관리
- **당첨 확인**: 저장된 번호의 당첨 여부 확인
- **통계 보기**: 구매 및 당첨 통계 확인
- **제외번호 추천**: 피해야 할 번호 추천

## 📊 DNN 모델 구조

### 특성 (Features)
1. **과거 번호 빈도**: 최근 N회차의 각 번호별 출현 빈도
2. **통계적 특성**: 합계, 홀짝 비율, 고저 비율의 평균/표준편차
3. **최근 패턴**: 직전 회차의 번호 조합
4. **연속성 패턴**: 연속된 번호의 개수
5. **비율 특성**: 홀짝 비율, 고저 비율

### 모델 아키텍처
```
입력층 → Dense(256) → Dropout(0.3) → Dense(128) → Dropout(0.3) 
→ Dense(64) → Dropout(0.2) → Dense(32) → 출력층(6)
```

### 훈련 설정
- **손실 함수**: Mean Squared Error (MSE)
- **옵티마이저**: Adam (학습률: 0.001)
- **정규화**: Dropout 레이어로 과적합 방지
- **콜백**: Early Stopping, Learning Rate Reduction

## 🤖 AI 패턴 학습 모델 구조

### 특성 (Features) - 60개
1. **번호별 빈도 특성 (45개)**: 각 번호(1-45)의 출현 빈도
2. **홀짝 비율 특성 (2개)**: 홀수 개수, 짝수 개수
3. **합계 통계 특성 (3개)**: 최소값, 최대값, 평균
4. **연속번호 패턴 특성 (5개)**: 평균, 최대, 최소, 연속번호 있는 회차 수, 2개 이상 연속인 회차 수
5. **구간별 분포 특성 (5개)**: 1-10, 11-20, 21-30, 31-40, 41-45 구간별 번호 분포

### 클러스터링
- **알고리즘**: K-means 클러스터링
- **특성 정규화**: StandardScaler
- **클러스터 수**: 사용자 설정 (기본: 5개)

### 머신러닝 모델
1. **RandomForest**: 앙상블 기반 분류 모델
2. **GradientBoosting**: 부스팅 기반 분류 모델

### 훈련 설정
- **데이터 분할**: 80% 훈련, 20% 테스트
- **교차 검증**: 5-fold cross validation
- **평가 지표**: 정확도 (Accuracy)
- **모델 저장**: Pickle 형식으로 저장

## 📈 성능 지표

### 평가 지표
- **MSE (Mean Squared Error)**: 예측값과 실제값의 평균 제곱 오차
- **MAE (Mean Absolute Error)**: 예측값과 실제값의 평균 절대 오차
- **정확도**: 예측된 번호 중 실제 당첨번호와 일치하는 비율

### 신뢰도
- 각 예측에 대한 0-1 범위의 신뢰도 점수
- 개별 번호별 신뢰도와 전체 조합의 평균 신뢰도 제공
