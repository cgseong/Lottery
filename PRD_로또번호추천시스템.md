# 로또 번호 추천 시스템 PRD (Product Requirements Document)

## 1. 프로젝트 개요

### 1.1 프로젝트명
**로또 번호 추천 시스템 (Lotto Number Recommendation System)**

### 1.2 프로젝트 목적
- 로또 당첨번호의 패턴을 분석하여 사용자에게 번호 추천 서비스 제공
- 다양한 분석 방법(DNN, AI 패턴 학습, 통계 분석)을 통한 종합적인 번호 추천
- 사용자 친화적인 인터페이스로 로또 번호 선택의 어려움 해소

### 1.3 타겟 사용자
- 로또 구매자
- 로또 번호 분석에 관심이 있는 사용자
- 데이터 기반 의사결정을 선호하는 사용자

## 2. 기능 요구사항

### 2.1 핵심 기능

#### 2.1.1 당첨번호 분석
- **기능**: 과거 당첨번호의 통계적 분석
- **세부 기능**:
  - 번호별 출현 빈도 분석
  - 홀짝 비율 분석
  - 구간별 분포 분석
  - 연속번호 패턴 분석
  - 합계 범위 분석

#### 2.1.2 번호 추천
- **기능**: 다양한 방법을 통한 번호 조합 추천
- **세부 기능**:
  - 통계 기반 추천
  - DNN(Deep Neural Network) 기반 추천
  - AI 패턴 학습 기반 추천
  - 사용자 지정 제외번호 반영

#### 2.1.3 제외번호 관리
- **기능**: 사용자가 제외하고 싶은 번호 관리
- **세부 기능**:
  - 제외번호 추가/제거
  - 제외번호 목록 조회
  - 제외번호 초기화
  - 제외번호 기반 추천

#### 2.1.4 당첨번호 수집
- **기능**: 동행복권 사이트에서 최신 당첨번호 자동 수집
- **세부 기능**:
  - 최신 회차 확인
  - 지정 범위 당첨번호 수집
  - CSV 파일 자동 저장
  - 중복 데이터 방지

### 2.2 고급 기능

#### 2.2.1 DNN 기반 예측
- **기능**: 딥러닝 모델을 통한 번호 예측
- **세부 기능**:
  - TensorFlow/Keras 기반 신경망 모델
  - 과거 데이터 학습
  - 예측 확률 계산
  - 모델 성능 평가

#### 2.2.2 AI 패턴 학습
- **기능**: 머신러닝을 통한 패턴 분석 및 예측
- **세부 기능**:
  - 회차 그룹핑 분석
  - 클러스터링 기반 패턴 분류
  - RandomForest, GradientBoosting 모델
  - 높은 확률 조합 예측

#### 2.2.3 시스템 모니터링
- **기능**: 시스템 상태 및 패키지 설치 상태 확인
- **세부 기능**:
  - 필수 패키지 설치 상태 확인
  - 설치 가이드 제공
  - 시스템 정보 표시

- **언어**: Python 3.7+
- **운영체제**: Windows, macOS, Linux
- **의존성 관리**: pip, requirements.txt
